## 线性表：某类元素的一个集合，还记录着元素之间的一种顺序关系
* 顺序表，将元素顺序地存放在一块连续的存储区里，存储顺序
* 链表，将元素存放在通过链接构造起来的一系列存储块中，在每一个节点（数据存储单元）存放下一个节点的地址
    * 单链表
    * 双向链表
    * 单向循环链表
> python中变量标识的本质：
a,b = b,a,保存对象的地址，在python中一切皆对象，a管一个地址，指向啥都行
